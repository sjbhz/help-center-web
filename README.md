# help-center-web

vue3 + pinia + wangEditor + element-plus 实现可配置化帮助中心
1. 按导航区-菜单区-内容区结构；
2. 支持帮助中心的菜单区支持新增、编辑、删除；
3. 菜单区编辑/预览切换；
4. 内容区编辑/预览切换；
5. 内容区支持图片/视频的上传
6. 另有tinyMCE、ckEditor、vmdEdiotr多个编辑器试用
